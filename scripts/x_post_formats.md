# X/Twitter 发布格式

## 短推 (<280 字)

```
{核心观点 — 一句扎心的结论}

{展开 2-3 句 — 为什么、举例、反例}

{收尾 — 留给读者自己想的空间}
```

## 中推 (280-800 字)

```
{核心观点 — 一句扎心的结论}

{论证 1 — 一个角度}
{论证 2 — 另一个角度}
{论证 3 — 更深一层}

{行动建议或反问}
```

## 长线程 🧵 (5-7 条)

```
1/  {扎心结论 hook}
    必须让人想点开看第 2 条

2/  {论证角度 1}
    讲一个具体场景或例子

3/  {论证角度 2}
    换个角度再扎一刀

4/  {反常识或内幕}
    大部分人不知道的

5/  {行动建议}
    知道了怎么办

6/  {收尾}
    关注我，持续输出商业认知。
    我是 Arthur，在苏里南用 AI 做生意的创业者。
```

## 发布检查清单

- [ ] 读出声来，像不像人说的话？
- [ ] 有没有废话可以删？
- [ ] 第一句能让人停下来吗？
- [ ] 如果我是读者，我会想转发/评论吗？
- [ ] [AI-native] 标注的，违和吗？
